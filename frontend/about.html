About AI MedAssist

An AI-powered healthcare web application that predicts diseases based on symptoms using Machine Learning models, designed to bridge the gap between clinical precision and human-centric empathy.

lightbulbProject Objective

AI MedAssist is designed to make healthcare accessible to everyone, especially in rural and underserved regions where medical professionals are scarce. Using advanced ML algorithms (Random Forest + XGBoost ensemble), the system predicts diseases from symptoms with high confidence, provides precautionary advice, and supports multilingual voice input.

  • Predict diseases from user-entered symptoms with 98%+ accuracy
  • Provide confidence scores for transparent AI-driven diagnosis
  • Support voice input for hands-free symptom entry
  • Offer precise healthcare guidance in English
  • Enable emergency ambulance dispatch
  • Maintain complete diagnosis history for users

schemaML Pipeline

The disease prediction pipeline uses an ensemble approach combining Random Forest (40% weight) and XGBoost (60% weight) classifiers trained on a curated dataset of 26 diseases and 100+ symptoms. The system:

  • Converts symptom input into binary feature vectors
  • Runs predictions through both RF and XGBoost models
  • Computes weighted ensemble probabilities for each disease
  • Returns the top prediction with confidence score and alternatives
  • Determines severity level and maps appropriate precautions
